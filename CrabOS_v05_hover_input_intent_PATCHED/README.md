🦀 Intent Tracer

ADS-powered browser extension for stabilising human intent before AI or action

🚀 What it is

Intent Tracer is a browser extension that observes:

where you are (page context)
what you look at (hover trail)
what you type (input)

…and turns that into a stable, usable intent.

Instead of:

messy thought → bad AI output

you get:

messy thought → stabilised intent → better results

🧠 Core idea

Most tools optimise the AI.

This tool optimises you before the AI.

Human (unstable)
→ Intent Tracer (ADS: Align → Differentiate → Stabilise)
→ Clean input
→ AI / Website performs better

🛠️ Installation
Clone repo
Open Chrome → Extensions
Enable Developer Mode
Load unpacked → select project folder
⚙️ Usage
Hover over content → builds context
Type in any field → triggers stabilisation
Open 🦀 panel → see intent trace
Click Apply Rewrite if needed
⚠️ Notes
No external APIs used
Runs fully in-browser
No data is sent anywhere
